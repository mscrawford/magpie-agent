# Project Management Directory

**⚠️ This directory is for documentation project maintainers only.**

If you're a MAgPIE developer looking to use the AI documentation, **you don't need anything in this folder**. Go back to the parent directory and see:
- `README.md` — Project overview
- `AGENT.md` — AI agent instructions
- `modules/` — Module documentation
- `core_docs/` — Architecture documentation
- `audit/` — Internal iteration artifacts (validation rounds, pipeline audits, lessons)

---

## What's in this folder?

This directory contains 4 files supporting the AI documentation project:

| File | Purpose |
|---|---|
| `README.md` | This file (explains folder contents) |
| `sync_log.json` | MAgPIE sync state — tracks last sync commit and modules reviewed. Read by `agent/helpers/session_startup.md` and the `/sync`, `/update`, `/validate` commands. |
| `version_pins.json` | Per-clone version pin snapshot — regenerated by `scripts/sync_magpie4_clone.py`. Read by `agent/helpers/magpie4_reference.md`. (Locally-generated; .gitignored.) |
| `archive/CURRENT_STATE.v1.0_frozen_2026-03-07.json` | **FROZEN v1.0 snapshot** (2026-03-07). Was the SSOT through v1.0; retired by R6 audit (Cluster A), relocated to `archive/` by R25 follow-up. Kept for history. Numbers inside (10 helpers, 6 commands, 13-round flywheel) are the 2026-03-07 reality. |

## Quick Start for Project Maintainers

If you're continuing work on the documentation project:

1. Read `audit/get_under_control_plan.md` for the current phased plan (if active).
2. Check `audit/validation_rounds.json` tail entries for the most recent flywheel rounds.
3. Check `audit/pipeline_audit_rounds.json` tail for structural audits.
4. Check `audit/next_session_plan.md` for any open one-off work items.
5. `git log --oneline -20` for recent commits.

(Do NOT edit `archive/CURRENT_STATE.v1.0_frozen_2026-03-07.json` — it is a frozen snapshot.)

- Live state lives in `audit/`, not here.
- The `audit/` directory is internal iteration — there is no longer an external user-submission inbox (the `/feedback` command was removed 2026-05-24, commit 327116c).
