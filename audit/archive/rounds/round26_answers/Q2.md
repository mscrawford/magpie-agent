QUESTION: What exactly enters MAgPIE's global objective function via module 11? Name the objective variable and the module-11 equation (q11_*) that aggregates costs, and list at least four vm_cost_* terms it sums, identifying which module owns each term. Cite equation names and file:line for the aggregation.

---

## Answer

### Objective variable

`vm_cost_glo` (global total cost, mio. USD17MER/yr) is MAgPIE's objective function variable. The solver **minimizes** it. It is declared in `modules/11_costs/default/declarations.gms:9`.

### Two-tier aggregation

Module 11 (`default` realization, the only realization) implements exactly two equations:

**`q11_cost_glo`** (`equations.gms:10`) — the top-level equation that feeds the solver objective:

```gams
q11_cost_glo .. vm_cost_glo =e= sum(i2, v11_cost_reg(i2));
```

This sums the intermediate regional variable `v11_cost_reg(i)` over all active regions (`i2`).

**`q11_cost_reg`** (`equations.gms:15-47`) — the workhorse that aggregates 32 terms (31 positive cost terms + 1 negative reward term) into `v11_cost_reg(i2)`. This is the equation that lists every `vm_cost_*` contributor.

So the chain is:

```
vm_cost_* (from 27 modules) --> q11_cost_reg --> v11_cost_reg(i) --> q11_cost_glo --> vm_cost_glo (minimized by solver)
```

### The one negative term

`vm_reward_cdr_aff(i)` from **M56** (GHG Policy) enters with a **negative sign** (`equations.gms:27`):

```gams
- vm_reward_cdr_aff(i2)
```

This represents revenue from afforestation-based carbon dioxide removal under a carbon price, reducing net costs.

### Full term count in q11_cost_reg

`q11_cost_reg` (`equations.gms:15-47`) sums **32 terms**: 31 positive cost components from 27 source modules, plus 1 negative reward term (`vm_reward_cdr_aff`). The complete list of all 27 contributing modules with their variables is in `modules/module_11.md` Section 17.2.
