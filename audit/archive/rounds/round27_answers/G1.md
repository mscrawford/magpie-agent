### Equations defined in equations.gms

Module 14 defines exactly **2 equations** in `equations.gms`:

#### 1. `q14_yield_crop` — Crop Yield Calculation
**File:** `equations.gms:14-16` (cited range in docs: 14-20)

```gams
q14_yield_crop(j2,kcr,w) ..
 vm_yld(j2,kcr,w) =e= sum(ct,i14_yields_calib(ct,j2,kcr,w)) *
                         vm_tau(j2,"crop") / sum((cell(i2,j2), supreg(h2,i2)), fm_tau1995(h2));
```

Calculates cellular crop yields by scaling the calibrated baseline yield (`i14_yields_calib`) by the ratio of the current technological-change factor (`vm_tau`, from module 13) to the 1995 baseline (`fm_tau1995`). Output: `vm_yld(j,kcr,w)` in tDM/ha/yr.

#### 2. `q14_yield_past` — Pasture Yield Calculation
**File:** `equations.gms:35-39` (cited range in docs: 24-39)

```gams
q14_yield_past(j2,w) ..
 vm_yld(j2,"pasture",w) =e=
 sum(ct,(i14_yields_calib(ct,j2,"pasture",w))
 * sum(cell(i2,j2),pm_past_mngmnt_factor(ct,i2)))
 * (1 + s14_yld_past_switch*(sum((cell(i2,j2), supreg(h2,i2)), pcm_tau(j2, "crop")/fm_tau1995(h2)) - 1));
```

Calculates pasture yields using the calibrated baseline, an exogenous pasture management factor (`pm_past_mngmnt_factor` from module 70), and partial spillover from crop technological change. Key distinction: uses `pcm_tau` (previous time step's tau) rather than `vm_tau` (current), representing delayed transfer of crop intensification to pasture. The spillover magnitude is set by `s14_yld_past_switch` (default 0.25, i.e. 25% spillover; `input.gms:20`).
