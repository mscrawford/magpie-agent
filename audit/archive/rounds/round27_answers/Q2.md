## Module 53 Methane Computation

**Source**: `modules/module_53.md` (verified against `../modules/53_methane/ipcc2006_aug22/*.gms`)

---

### Default Realization

The default realization is **`ipcc2006_aug22`**, confirmed in `config/default.cfg`:
`cfg$gms$methane <- "ipcc2006_aug22"` (module_53.md, module structure section).

The alternative realization `off` disables all CH4 calculation.

---

### Emission Variable

All four equations write to the single interface variable **`vm_emissions_reg(i2, emis_source, "ch4")`**, declared in Module 56 (GHG Policy). The four `emis_source` elements populated by Module 53 are: `ent_ferm`, `awms`, `rice`, `resid_burn` (module_53.md, sets section; `sets.gms:22-23`).

---

### The Four Equations

#### 1. q53_emissionbal_ch4_ent_ferm (equations.gms:21-29)

Calculates CH4 from enteric fermentation in ruminants.

Formula:
```gams
vm_emissions_reg(i2,"ent_ferm","ch4") =e=
  1/55.65 * (
    sum(k_conc53, vm_feed_intake(i2,"livst_rum",k_conc53) * fm_attributes("ge",k_conc53) * 0.03)
    + sum(k_conc53, vm_feed_intake(i2,"livst_milk",k_conc53) * fm_attributes("ge",k_conc53) * 0.065)
    + sum((k_noconc53,k_ruminants53), vm_feed_intake(i2,k_ruminants53,k_noconc53)
          * fm_attributes("ge",k_noconc53) * 0.065)
  ) * (1 - sum(ct, im_maccs_mitigation(ct,i2,"ent_ferm","ch4")));
```

**Activity driver from Module 70**: `vm_feed_intake(i2, kap, kall)` (declared in Module 70, `fbask_jan16/declarations.gms:18`). Module 53 uses the subset for ruminants only:
- `vm_feed_intake(i2,"livst_rum",k_conc53)` - beef cattle on concentrates
- `vm_feed_intake(i2,"livst_milk",k_conc53)` - dairy cattle on concentrates
- `vm_feed_intake(i2,k_ruminants53,k_noconc53)` - all ruminants on non-concentrates (pasture, fodder, residues)

Feed is converted to CH4 via: feed intake (tDM) x gross energy `fm_attributes("ge",kall)` (GJ/tDM) x Ym conversion rate, then divided by 55.65 MJ/kg (CH4 energy content) to yield tCH4. Ym rates are 0.03 for beef on concentrates (higher digestibility) and 0.065 for dairy and all ruminants on non-concentrates (equations.gms:18-20).

MACC technical mitigation `im_maccs_mitigation(ct,i2,"ent_ferm","ch4")` from Module 57 applied as a multiplicative reduction (equations.gms:29).

#### 2. q53_emissionbal_ch4_awms (equations.gms:48-52)

Calculates CH4 from anaerobic manure decomposition in confinement systems.

Formula:
```gams
vm_emissions_reg(i2,"awms","ch4") =e=
  sum(kli, vm_manure(i2, kli, "confinement", "nr")
      * sum(ct, f53_ef_ch4_awms(ct,i2,kli)))
  * (1 - sum(ct, im_maccs_mitigation(ct,i2,"awms","ch4")));
```

Activity driver is `vm_manure(i2, kli, "confinement", "nr")` from Module 55 (AWMS): nitrogen in confinement manure (tN/yr). The emission factor `f53_ef_ch4_awms(t_all,i,kli)` (tCH4/tN) is read from `f53_EFch4AWMS.cs4` (input.gms:8-11). Confinement-only (grazing manure excluded as aerobic decomposition yields negligible CH4). MACC mitigation applied (equations.gms:52).

#### 3. q53_emissionbal_ch4_rice (equations.gms:59-63)

Calculates CH4 from anaerobic decomposition in flooded rice paddies.

Formula:
```gams
vm_emissions_reg(i2,"rice","ch4") =e=
  sum((cell(i2,j2),w), vm_area(j2,"rice_pro",w)
      * sum(ct, f53_ef_ch4_rice(ct,i2)))
  * (1 - sum(ct, im_maccs_mitigation(ct,i2,"rice","ch4")));
```

**Rice area driver**: `vm_area(j2,"rice_pro",w)` (ha) from Module 30 (Croparea) or Module 17 (Production), summed over simulation cells (`j2`) belonging to region `i2` via the `cell(i2,j2)` mapping, and over water management types `w` (rainfed and irrigated). The emission factor `f53_ef_ch4_rice(t_all,i)` (tCH4/ha/yr) is regional only (does NOT distinguish irrigated vs. rainfed), read from `f53_EFch4Rice.cs4` (input.gms:16-19). MACC mitigation applied (equations.gms:63).

#### 4. q53_emissions_resid_burn (equations.gms:70-72)

Calculates CH4 from incomplete combustion of crop residues.

Formula:
```gams
vm_emissions_reg(i2,"resid_burn","ch4") =e=
  sum(kcr, vm_res_ag_burn(i2,kcr,"dm")) * s53_ef_ch4_res_ag_burn;
```

Activity driver is `vm_res_ag_burn(i2,kcr,"dm")` from Module 18 (Residues). Emission factor `s53_ef_ch4_res_ag_burn = 0.0027 tCH4/tDM` is a global scalar (input.gms:24-26), based on IPCC 2019 Refinement Eq. 2.27. This equation has **no MACC mitigation term** - mitigation occurs upstream by reducing burning itself (Module 18 decision), not by changing emissions per unit burned (module_53.md, equations.gms:70-72 note).

### How Emissions Feed GHG Pricing in Module 56

Source: `modules/module_56.md` (default realization `price_aug22`).

The CH4 entries in `vm_emissions_reg` are annual (recurring) emissions. Module 56 picks them up via:

**Step 1 - q56_emis_pricing (equations.gms:15-17)**:
```gams
q56_emis_pricing(i2,pollutants,emis_annual) ..
  v56_emis_pricing(i2,emis_annual,pollutants) =e=
    vm_emissions_reg(i2,emis_annual,pollutants);
```
The four CH4 sources (`ent_ferm`, `awms`, `rice`, `resid_burn`) all fall under `emis_annual` (recurring emissions), so `v56_emis_pricing(i2,emis_source,"ch4")` = the corresponding `vm_emissions_reg` value.

**Step 2 - q56_emission_cost_annual (equations.gms:29-33)**:
```gams
q56_emission_cost_annual(i2,emis_annual) ..
  v56_emission_cost(i2,emis_annual) =e=
    sum(pollutants,
        v56_emis_pricing(i2,emis_annual,pollutants) *
        sum(ct, im_pollutant_prices(ct,i2,pollutants,emis_annual)));
```
Each CH4 source's emissions (Tg CH4/yr) are multiplied by `im_pollutant_prices(ct,i2,"ch4",emis_source)` (USD17MER/Mg), yielding `v56_emission_cost` in mio. USD17MER/yr (because 1 Tg = 10^6 Mg). The GHG price scenario is controlled by `c56_pollutant_prices` (default `R34M410-SSP2-NPi2025`); which sources are actually priced is controlled by `c56_emis_policy` (default `reddnatveg_nosoil`).

These costs enter Module 11 (Costs) and thus the MAgPIE objective function, creating an incentive for the optimizer to reduce CH4-generating activities or adopt MACC mitigation measures.

**GWP note**: CH4 is priced directly as a gas (not pre-converted to CO2-eq inside Module 53). Conversion to CO2-eq equivalent pricing occurs implicitly through the price schedule loaded for `"ch4"` in `im_pollutant_prices`, which is calibrated using AR4 GWP = 25 (100-year) for consistency with the MACC curves from Module 57.
